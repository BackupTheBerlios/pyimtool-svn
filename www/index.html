PyImtool

PyImtool is a Mac OS X native IRAF image server.

This means that you can use PyImtool together with IRAF to display and interact with astronomical data without the need for an X11 server.
PyImtool is developed mainly in Python with some Objective-C here and there to improve performance. It uses PyObjC, the Python-Objective-C bridge.

PyImtool is free (as in freedom) for anybody to use.

There is no release yet. Stay tuned.
